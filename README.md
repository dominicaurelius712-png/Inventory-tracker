## 🚀 Live Application
**Access the live tracker here:** 🔗 https://inventory-tracker-smallbussiness.netlify.app/
## 🛠️ Technology Stack
 * **Frontend:** Pure HTML5, Vanilla JavaScript, Tailwind CSS.
 * **Storage:** IndexedDB (Local Browser Storage).
## ⚠️ Important Note on Data Storage
This application currently uses **IndexedDB** for an offline-first experience.
 * **Data is stored locally** in your current web browser.
 * **No cloud sync:** If you clear your browser cache, or switch to a different device (like moving from your laptop to your phone), your inventory data will **not** be available.
 * Always export your data manually if you plan to clear your browser
